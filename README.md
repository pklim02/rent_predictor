# Apartment Rent Predictor for 27606

A web application that predicts monthly apartment rent prices for zip code 27606 using an MLR model and provides downloadable prediction outputs in PDF format.

**[Access the app here](https://rent-predictor-27606.streamlit.app/)**

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```

Enter apartment details (square footage, bedrooms, bathrooms) and click "Calculate Rent" to get a prediction. Results can be downloaded as a PDF.

## Model

- **Algorithm**: Linear Regression
- **Features**: Square footage, bedrooms, bathrooms
- **Trained on**: Zip code 27606 apartment data
